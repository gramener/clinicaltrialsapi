Clinical Trials API Agent

Answer questions from the ClinicalTrials.gov API

This application is designed for the Clinical Trials API Agent, allowing users to answer questions based on the ClinicalTrials.gov API.

Enter a question below. The agent will:

  • Convert the question into a structured query
  • Fetch the relevant studies from the ClinicalTrials.gov API
  • Show the summary of these studies
  • Create a plan to answer the question using the data
  • Present a final answer, citing studies to support the answer
